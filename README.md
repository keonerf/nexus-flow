# ProtoPulse
**From Gerber to Prototype in 60 Minutes.**

ProtoPulse (formerly NexusFlow) represents the future of hardware prototyping. Our flagship system, **ProtoBlock-1**, integrates isolation milling, conductive paste dispensing, pick-and-place, and reflow soldering into a single desktop unit.

## Project Structure
- **Frontend**: React + Vite + Framer Motion
- **Styling**: Glassmorphism / Industrial Dark UI

## Getting Started
```bash
npm install
npm run dev
```

## Deployment
Deployed on Vercel.
